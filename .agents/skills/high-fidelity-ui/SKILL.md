# High-Fidelity UI Architect

## Description
Expertise in building high-fidelity React interfaces with persistent headers, canvas particle systems, and adaptive light/dark mode components. Triggers on requests for premium frontend development and UI consistency.

## Instructions
1. **Header Consistency**: Always import the existing `<Header />` component. Do not recreate navigation logic.
2. **Hero Implementation**: Apply a full-bleed Hero section with centered typography. Integrate the particle system background for interactive depth.
3. **Pill-Button Styling**: Use pill-shaped buttons. Apply glassmorphism for secondary actions and high-contrast solid fills for primary CTAs.
4. **Adaptive Themes**: Ensure all components use CSS variables that respond to both Light and Dark modes.
5. **Iconography**: Strictly use the project's designated icon pack for all visual indicators.
6. **Bento Grid Layout**: Organize secondary content into modular, asymmetrical grids with 12-24px rounded corners.
